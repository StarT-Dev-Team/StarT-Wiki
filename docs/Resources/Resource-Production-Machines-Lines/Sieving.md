# A guide to Sieving

## The early game sieving

In the early game \(tiers <ULV>**ULV**</ULV> and <LV>**LV**</LV>\) you may be primarily using Ex Nihilo to get ores and resources from sieving various loose materials, them beeing:

- Gravel
- Sand
- Dust
- Crushed Blackstone
- Mud
- Dirt
- Any Leaves
