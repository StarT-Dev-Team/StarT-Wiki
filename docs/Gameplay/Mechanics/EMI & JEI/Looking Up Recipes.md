# Basic features of EMI

## Looking up recepies

To look up a recepie you need to press **R** on any item you can hover over, pressing **Left Click** in the sidebar will also work but in the inventory it will select the item.

## Looking up uses

To look up uses for an item you need to press **U** on any item you can hover over, pressing **Right Click** in the sidebar will also work but in the inventory it will select the item.
