# Star Technology Zeta Changelog:

## Zeta 1 Changelog:

**Release Date:** 23/04/202

**Additions:**

- Updated Quest Book
- Descriptions on all Main Quest and more
- Revamped Pages
- All custom multiblocks have tooltips and some GT ones too
- Machines now show whether they are capable of perfect overclock or subticks, and if they accept parallel hatches.
- Depth added to ASG line
- Note from Devs: “The Ancient Stargate Line was fairly bland and just more spam. This was not unique or engaging gameplay, so while the additions may not be perfect, they should make it more interesting”
- Ultimate Tier Multiblocks
- Cyclonic Sifter, Injection Mixer, Manifold Centrifuge, Molten Destabilizer
- Resource Gen QoL
- Cobaltite Sludge
- Geode Targeting
- EV Molten Ore Mixture Recipe
- Brackish Water
- Note from Devs: “There were some resource bottlenecks that were unintended in Epsilon such as easy cobalt, wait time for geodes, being stuck in EV, and poor sources of Sea Water. These additions should help alleviate those painpoints”
- Early Game QoL Machines
- Steam Ore Factory (Courtesy of 20Jonas02)
- Latex Plantation
- Advanced Composter
- Multiblock Viewer for CSG and power helper system
- More Parallel Hatches

**Changes:** 

- LuV+ Superconductor Recipes adjusted and amperage caps changed
- Made less recursive
- Line Reworks
- Akreyrium
- Note from Devs: “The system felt bland and surface level for a post-Gate line so was revamped to have more complexity and uniqueness”
- Nether Stars
- Note from Devs: “Nether Stars were in a weird place being necessary but feeling like a weird system. Inspired from crystal chips and with their own unique twist this new system should be more engaging and lead to better results. Non-MA methods made more viable as well”
- Mechanical Sieve Nerf
- Note from Devs: “The fast scaling of the sieve’s production of resources made it possible to skip a lot of the progression system, so it has been nerfed to better align with other resource generation methods.”
- Fixed some lines not looping properly
- Stargate Tier Materials Made less time consuming
- Stellarium (50% faster), Runicalium (40% faster), Weapon Grade Naq (~11% faster), more
- Note from Devs: “The endgame shouldnt comprise of so much ‘waiting game’ so this should help reduce that”
- Pure Netherite and Netherite swapped Coil/Voltage Requirements
- UHV+ Component Parts made to be less consumptive of prior tiers
- Note from Devs: “A slight miscalculation exponentially increased the demand for UHV components and was adjusted to be more reasonable”
- Coordinate Crystals are more Interactive
- Auxiliary MK I Fusion Reactor Adjustments
- Casings Cost reduced
- Uses ANY Absolute Parallel Hatch
- Abydos Resource Skips Rebalancing
- Exotic Gas Abydos made to be more of a generalist
- Bacteria output increased and voltage decreased
- Dynamo Adjustments
- Diamonds Nerfed (500,000FE => 300,000FE)
- All Systeam Steams Adjusted
- Note from Devs: “Dynamo spam into MK III’s was not something intended, these adjustments should reduce their viability at higher ends as well as rewarding players for setting up systems like Diatrons”
- Various other fixes and adjustments
- Disabled ME pattern buffer due to instability
- Disabled World Accelerators due to instability
- Visual Fixes
- Bug Fixes
- Performance Improvements
- More

**Mod Changes:**
